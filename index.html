Introduction

Welcome to my GitHub page. As a software specialist, I sometimes tinker with a variety of technologies in areas that are unrelated to my core competencies. Some of it ends up on GitHub so you will find here the result of that tinkering. All code, including the HTML code for this web page, is released under an open source license so feel free to copy, modify and fork anything you find on those pages.

GitHub Projects

  • nhtg12

    Prototype for Rewired State National Hack The Government Day, 2012

    NHTG12 was a lot of fun. This is my entry to the competition. It didn't win any prize but I enjoyed creating it and I learnt a thing or two in the process.

  • octave-fractals

    Scripts to create Mandelbrot and Julia fractals in Octave

    I've always been fascinated by fractal geometry and the images of the Mandelbrot and Julia sets. So after reading an article on the subject in Free Software Magazine, I decided to have a go at doing fractal images with Octave. In the process, I discovered quite a bit about fractals.

  • bike

    Bicycle accessories for 3D printing

    This project contains 3D description files written in the openscad langauge. They are meant to be compiled to the STL format in order to be printed by a 3D printer.
